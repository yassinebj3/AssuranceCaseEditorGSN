Constraints

Not every combination of source type, association type and target type is valid. Every ill-formed association will be deleted upon creation. Similarly, if the user modifies the source or target of an association in such a way that the association becomes invalid, the modification will be ignored and the association will keep its former source and target.

This section of the GSN plug-in manual describes every valid combination of source type, association type and target type.

InContextOf:

  • Goal -> Context
  • Goal -> Assumption
  • Goal -> Justification
  • Goal -> AwayGoal
  • Goal -> AwayContext
  • Goal -> Module
  • Strategy -> Context
  • Strategy -> Assumption
  • Strategy -> Justification
  • Strategy -> AwayGoal
  • Strategy -> AwayContext
  • Strategy -> Module

The same combinations are valid with associations of type OptionnalInContextOf and N-aryInContextOf.

SupportedBy:

  • Goal -> Goal
  • Goal -> Strategy
  • Goal -> Solution
  • Goal -> AwayGoal
  • Goal -> AwaySolution
  • Goal -> Module
  • Goal -> ContractModule
  • Strategy -> Goal
  • Strategy -> AwayGoal
  • Strategy -> AwaySolution
  • Strategy -> Module
  • Strategy -> ContractModule

The same combinations are valid with associations of type OptionnalSupportedBy and N-arySupportedBy.
